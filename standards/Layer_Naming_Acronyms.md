# StructureCraft — Layer Naming Convention & Acronyms

A glossary of the tokens used in the Branch / StructureCraft layer standard.
Naming convention: `SC_<MATERIAL>_<COMPONENT>_<SUBTYPE>` (strict, underscore-delimited).

---

## Prefixes (who / what owns the layer)

| Token | Meaning | Notes |
|-------|---------|-------|
| `SC` | StructureCraft | Manual / user-authored layers |
| `BR` | Branch | Auto-generated by Branch (in-house software) |
| `CE` | Connection Entities | The connection library (formerly "block" / "collection" library) |
| `LB` | Library | The connection-entity library itself |

## Containers / groups

| Token | Meaning |
|-------|---------|
| `BUILDING` | 3D model container for building elements |
| `L01`…`L05` | Building level / storey |
| `LIBRARY` | Connection-entity (CE) library container |
| `REF` → `REF_ARCH` / `REF_STRUCT` / `REF_MECH` | Reference models from other disciplines |
| `USER` | Manual annotation layers (`SC_ANN_*`) |
| `AUTO` | Branch-generated annotation layers (`BR_ANN_*`) |
| `SC-LAYOUT` | Sheet / layout layers |

## Materials

| Token | Meaning |
|-------|---------|
| `GLM` | Glulam (also seen as `GL` / `GLULAM`) |
| `CLT` | Cross-Laminated Timber |
| `DLT` | Dowel-Laminated Timber |
| `STL` | Steel |
| `BLK` | Blocking |
| `TIMBER` | Generic wood (any species/product; not product-specific) |

## Components

| Token | Meaning |
|-------|---------|
| `BEAM`, `COLUMN`, `POST`, `WALL`, `PANEL` | Primary members |
| `BILLET` | Glulam billet (raw blank / stock) |
| `TRUSS` → `TOPCHORD` / `BOTCHORD` / `WEB` | Truss members (bottom chord / web) |
| `PLATE` → `BASE` / `BENT` / `HANGER` / `KNIFE` / `Z` | Steel plate types |
| `ROD`, `ANGLE` | Steel elements |

## Machining objects (Branch cutting objects)

| Token | Meaning |
|-------|---------|
| `TOOL` | Tooling / machining layer group |
| `DAP` | Dap / notch — cutting object that removes material |
| `PEN` | Penetration — a dap that passes through the timber (e.g. holes in DLT decking) |
| `CHASE` | Routed channel / chase |
| `CUTTING` / `CUT` | Cut operation |
| `PRE_DRILL` | Pre-drilled holes |
| `OPENING` | Openings / voids |

## Connections & fasteners

| Token | Meaning |
|-------|---------|
| `CONN` → `BEARING` / `HANGER` | Connection type |
| `FAST` / `FASTENERS` | Fasteners |
| `STRONG_BACK` | Strongback (stiffening member, typically DLT) |
| `PLUG`, `EMBED`, `LEDGER`, `BRACE`, `SPACER`, `SHIM`, `NUT/WASHER` | Connection hardware / parts |

## Annotation & shop-drawing linework (`BR_LAYERSTACK`)

| Token | Meaning |
|-------|---------|
| `ANN` | Annotation |
| `DIM` / `TEXT` / `LEAD` | Dimension / text / leader |
| `LINE` → `CENTER` / `OUTLINE` / `PROFILE` / `HIDD_A…D` / `STITCH` | Linework (hidden-line weights A–D) |
| `HATCH` (`A` / `B` / `C`) | Hatch fills |
| `ELEM` → `LIFTPOINTS` / `FAST` / `ACOUSTIC` / `MEMBRANE` / `INFILLPARALLEL` / `INFILLPERP` / `STRONGBACK` / `SHEATH` | Element callouts |
| `VIEWS`, `RED`, `BLUE`, `TEMPORARY` | View markers / markup / temp |
| `LP` | Lifting points |
| `HIDD` | Hidden |

## Regions / sourcing (connection library)

| Token | Meaning |
|-------|---------|
| `NA` | North America |
| `EU` | Europe |
| `SCB` | StructureCraft Builders |
| `SITE` / `SHOP` | Field (on-site) / shop (fabrication) |
| `PREFAB` | Prefabricated |
| `HASSLACHER` | Timber supplier (note: `HASSALCHER` in the data is a typo) |

## Connection-library color legend (from S448)

Standard colors for the connection-entity library (`CE_LB`), applied in all three standards.

| Layer | Hex | Meaning |
|-------|-----|---------|
| `CE_STEEL_EU` | `#0080FF` blue | EU steel, installed by Hasslacher |
| `CE_STEEL_NA_SITE` | `#00D084` green | NA steel, installed on site |
| `CE_STEEL_NA_SHOP` | `#004080` dark blue | NA steel, installed in Abby shop |
| `CE_FASTENER_EU` | `#80FFFF` cyan | EU fasteners, installed by Hasslacher |
| `CE_FASTENER_NA_SITE` | `#80FF80` light green | NA fastener, installed on site |
| `CE_EMBED_NA` | `#C0C0C0` grey | NA embeds, installed on site by others |
| `CE_RUBBER_NA` | `#9C27B0` purple | NA rubber, installed on site |
| `CE_SHIM_NA` | `#FFFF80` yellow | NA plastic shim, installed on site |

## Material / object color palette (SC Colour Palette)

Applied to **model member layers** (`SC-OBJECTS` / bridge segments) that had **no color assigned**
(does not override the connection-library colors above).

| Material | Hex | Applied to |
|----------|-----|------------|
| Glulam | `#F3BA5C` | `*_GLM_*` members, `BILLET`, `RING_BEAM`, `LATH` |
| Steel | `#696969` | `*_STL_*` members, `STEEL`, `STRONG_BACK` |
| DLT / CLT floor panels | `#FFD072` | `*_DLT_*` / `*_CLT_*` panels |

Available in the palette but **not yet mapped** (pending decision): Glulam Dark `#BA7F41`,
DLT Light `#FEDF9F`, Concrete Topping Slab `#FFFFFF`, Steel Connections Opt1/2/3
(`#B80000` / `#0052D0` / `#C6C6C6`); and no-hex entries Glulam Intense, CLT/DLT Wall Dark, Fasteners.
